css 색상 속성

background-color: 배경색상
color: 글자색상
border-color: 테두리 색상
색상값: 색상이름 / 색상표(#2f34a1) / rgb(255,255,255)

배경 제어

background-image: 배경 이미지 지정

background-repeat:repeat-x(x축으로만 반복)
background-repeat:repeat-y(y축으로만 반복)
background-repeat:no-repeat(반복안함)
background-position:수평위치(left/center/right) 수직위치(top/center/bottom)

테두리 제어

- border-style(테두리 모양 설정 속성)
- border-color: (테두리색상)
- border-width: (테두리두께)
- border:2px dotted red => 테두리 두께 모양 색상을 한번에 설정
